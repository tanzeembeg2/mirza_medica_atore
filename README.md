# Mirza Medical Store Pharmacy ERP

## Run

1. Install dependencies:
   pip install -r requirements.txt
2. Start the app:
   python app.py
3. Open in browser:
   http://127.0.0.1:5000

## Login

- Username: admin
- Password: 1234

## Features

- Dashboard with sales, low stock, expiring stock, and trend chart
- Keyboard-first billing with GST and discount handling
- Inventory add, edit, delete, and Excel import/export
- Customer management
- Reports for recent sales and top-moving medicines
- Persistent file-backed data store in `pharmacy_data.json`
